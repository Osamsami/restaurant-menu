# Al Ibnis - Restaurant Menu

A static, responsive restaurant menu page featuring a dynamic order form.

## Live Demo
Check out the live deployment here: [https://restaurant-menu-six-sage.vercel.app/](https://restaurant-menu-six-sage.vercel.app/)

## Features
- **Menu Display:** Showcases menu items with attractive images and descriptions.
- **Order Form:** Allows users to select dishes and specify quantities.
- **Client-Side Validation:** Ensures required fields (like name and minimum quantity) are filled out properly before submission.
- **Order Summary:** Dynamically displays a confirmation of the order upon successful validation.
- **Responsive Design:** Adapts smoothly to mobile and desktop screens using modern CSS Flexbox and media queries.
- **Keyboard Accessibility:** Includes focus states and ARIA labels for improved navigation and screen reader support.

## How to Run
This is a static project with no build steps or installations required. 
Simply open `index.html` in any modern web browser to view the application.

## Project Structure
- `index.html`: The main structural layout and semantic markup of the page.
- `style.css`: The stylesheet containing the color palette, typography, responsive layout, and interactive states.
- `script.js`: The JavaScript file handling the order form validation and summary display.
- `images/`: Directory containing the local image assets for the menu items.
